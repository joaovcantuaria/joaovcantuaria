═══════════════════════════════════════════════════════════════════
🎨 DICAS PARA CRIAR/ESCOLHER AS IMAGENS
═══════════════════════════════════════════════════════════════════

FOTOGRAFIA:
- Use imagens de alta qualidade e resolução
- Mantenha consistência no estilo visual
- Prefira fotos reais do seu trabalho
- Evite imagens genéricas de banco de imagens se possível

CORES:
- As imagens devem harmonizar com a paleta preta/dourada/branca
- Considere aplicar filtros sutis para unificar o visual
- Evite imagens muito coloridas que conflitem com o design

COMPOSIÇÃO:
- Imagens do portfólio devem mostrar claramente o trabalho
- Evite texto excessivo nas imagens (use as descrições no HTML)
- Mantenha o foco no trabalho, não em elementos decorativos

BANCOS DE IMAGENS GRATUITOS (Se precisar de placeholders temporários):
- Unsplash (unsplash.com)
- Pexels (pexels.com)
- Pixabay (pixabay.com)

Busque por:
- "photography equipment"
- "creative workspace"
- "advertising campaign"
- "professional photography"
- "marketing strategy"

═══════════════════════════════════════════════════════════════════
🛠️ FERRAMENTAS PARA EDITAR/OTIMIZAR IMAGENS
═══════════════════════════════════════════════════════════════════

EDIÇÃO BÁSICA (Gratuitas):
- Canva (canva.com) - Browser, muito fácil
- Photopea (photopea.com) - Browser, similar ao Photoshop
- GIMP - Desktop, poderoso e gratuito
- Pixlr (pixlr.com) - Browser, simples

COMPRESSÃO (Reduzir tamanho sem perder qualidade):
- TinyPNG (tinypng.com) - Melhor para PNG
- Squoosh (squoosh.app) - Google, várias opções
- ImageOptim - Mac, automático
- JPEG Optimizer - jpegoptimizer.com

REDIMENSIONAMENTO:
- Bulk Resize Photos (bulkresizephotos.com)
- Simple Image Resizer (simpleimageresizer.com)
